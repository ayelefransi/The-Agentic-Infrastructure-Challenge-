# Agent Skills Directory

This directory contains the "Skills" available to the Chimera Workers.
Each subdirectory is a self-contained skill module.

## Structure
- `SKILL_NAME/README.md`: Contract definition (Input/Output).
- `SKILL_NAME/impl.py`: Implementation (optional, if local).
- `SKILL_NAME/mcp.json`: MCP Server config (if remote).
